## 1. 产品概述

面向政府应急办值班人员的突发事件回应辅助平台，重点处理事件发生后最初两小时的舆情研判与口径整理。系统通过结构化的事件录入、实时舆情快扫和智能回应草稿三大模块，帮助值班人员在信息不完整但必须先回应的场景下，快速补齐事实、研判公众质疑、组织回应口径，最终导出可供领导审签的回应文本。

- 核心价值：在夜间值班、信息碎片化的高压场景下，将"不知道说什么"的焦虑转化为"有据可依、有章可循"的回应流程
- 目标用户：政府应急办值班人员、宣传口径起草人员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 值班人员 | 管理员分配账号 | 事件录入、舆情查看、草稿编写与导出 |
| 审签领导 | 管理员分配账号 | 查看草稿、审批退回 |

### 2.2 功能模块

1. **事件录入页**：事件基本信息表单、处置清单生成、事实补齐提示
2. **舆情快扫页**：四渠道舆情分类展示、高频质疑提炼、情绪强度标注
3. **回应草稿页**：三段式模板生成、逐句编辑、导出审签

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 事件录入 | 事件基本信息 | 填写事件名称、地点、时间、涉及单位、事件类型、伤亡等级 |
| 事件录入 | 已确认事实 | 逐条录入已确认的事实描述，支持标注来源与确认时间 |
| 事件录入 | 暂不确定信息 | 逐条录入暂不确定的信息点，标记待核实状态 |
| 事件录入 | 处置清单 | 根据录入信息自动生成处置清单，提示需要补齐的事实、需要联系的单位、需要核实的传言 |
| 舆情快扫 | 媒体报道 | 展示主流媒体报道摘要、报道倾向标签（客观/质疑/煽动） |
| 舆情快扫 | 短视频评论 | 展示短视频平台评论高频词云与代表性评论，标注情绪强度（1-5级） |
| 舆情快扫 | 社区论坛 | 展示论坛帖子摘要与讨论热度，标注主要质疑方向 |
| 舆情快扫 | 投诉热线摘要 | 展示12345等热线投诉分类统计与典型诉求 |
| 舆情快扫 | 高频质疑看板 | 汇总四渠道高频质疑（如是否瞒报、救援是否迟缓、责任人是谁），以卡片形式展示 |
| 回应草稿 | 事实说明段 | 基于已确认事实自动生成事实说明文字，用户可逐句勾选/删除/改写 |
| 回应草稿 | 安抚表述段 | 提供多种安抚表述模板，用户可选用或改写 |
| 回应草稿 | 后续措施段 | 基于事件类型推荐后续措施模板，用户可选用或改写 |
| 回应草稿 | 导出审签 | 将编辑完成的草稿导出为文档，包含时间戳、值班人员信息，供领导审签 |

## 3. 核心流程

值班人员接到突发事件报告后，首先进入事件录入页填写基本信息、已确认事实和暂不确定信息。系统据此生成处置清单，提示需要补齐的事实。随后进入舆情快扫页，查看四渠道舆情动态和高频质疑，研判公众关注焦点。最后进入回应草稿页，基于事实和舆情生成三段式回应模板，值班人员逐句编辑后导出给领导审签。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰（#1B2838）+ 琥珀警告色（#F59E0B），传递专业、冷静的应急氛围
- 辅助色：浅灰底（#F8F9FA）、白色卡片、红色紧急标记（#DC2626）、绿色确认标记（#16A34A）
- 按钮风格：圆角矩形（8px），主按钮实心填充，次按钮描边，操作按钮带图标
- 字体：标题使用 Noto Serif SC 传达公文质感，正文使用 Noto Sans SC 保证可读性
- 布局风格：左侧固定导航栏 + 右侧内容区，卡片式模块布局，顶部进度指示条
- 图标风格：线性图标，统一2px描边，与文字大小协调

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 事件录入 | 事件基本信息 | 表单输入框、下拉选择、日期时间选择器、地图定位点 |
| 事件录入 | 已确认事实 | 可增删的事实卡片列表，每条含文本、来源标签、确认时间 |
| 事件录入 | 暂不确定信息 | 可增删的信息卡片列表，每条含文本、待核实状态标签、紧急程度 |
| 事件录入 | 处置清单 | 自动生成的待办清单，带优先级色标和勾选框 |
| 舆情快扫 | 渠道标签切换 | 四个渠道标签页切换，带图标和数量角标 |
| 舆情快扫 | 媒体报道 | 卡片列表，含媒体名称、摘要、倾向标签、时间 |
| 舆情快扫 | 短视频评论 | 词云可视化 + 代表性评论卡片，情绪强度色条 |
| 舆情快扫 | 社区论坛 | 热度排行列表，含帖子标题、讨论量、主要质疑方向标签 |
| 舆情快扫 | 投诉热线摘要 | 饼图分类统计 + 典型诉求卡片 |
| 舆情快扫 | 高频质疑看板 | 质疑卡片网格，每张含质疑内容、出现频次、情绪强度指示器 |
| 回应草稿 | 三段式编辑区 | 分段可折叠面板，每句前有勾选框，悬停显示编辑按钮 |
| 回应草稿 | 导出审签 | 底部固定工具栏，含预览、导出Word、打印按钮 |

### 4.3 响应式设计

- 桌面端优先（1920×1080为主要分辨率），适配1440px和1280px
- 平板端（768px-1024px）侧边栏可收起为图标模式
- 移动端暂不考虑，因值班场景以桌面为主
